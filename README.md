Build with
===========
 - Angular 5
 - ASP.NET Core 2 MVC 

Install and run
===========
 - insert client id and secret into server source
 - build and run server
 - npm install
 - insert document URN into webapp source
 - npm start
 - navigate to http://localhost:4500
